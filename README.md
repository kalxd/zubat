zubat
=====

![zubat](https://media.52poke.com/wiki/archive/d/da/20140405224319%21041Zubat.png)

这是一款实实在的html5解析库，仅作解析，不作html的修改、删除操作。
是[golbat](https://github.com/kalxd/golbat)的racket包装实现。
